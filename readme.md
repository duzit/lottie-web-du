# Lottie 介绍

## Lottie 适用于 Android, iOS, Web, React Native 和 Windows
### Lottie是一个用于Android，iOS，Web和Windows的库，用于解析使用Bodymovin导出为json的Adobe After Effects动画，并在移动设备和网络上呈现它们！
